# hoshino_MoeGoe

HoshinoBot 的 MoeGoe 模块

## 安装

安装依赖

``` shell
pip install -r requirements.txt
```

在 `HoshinoBot/hoshino/config/__bot__.py` 的 `MODULES_ON` 中加入 `hoshino_MoeGoe` 启用。

对于 `go-cqhttp` 需要安装 `ffmpeg`。

## 使用
